# Чеклист перед игрой

## За день до игры

- [ ] Проверить, что в комнате свободно **4×4 м** (убрать мебель от краёв)
- [ ] Набрать кубики: **минимум 25–30 штук** одного размера
- [ ] Напечатать / нарисовать **~30 карточек лута** (`03-items-and-loot.md`)
- [ ] Подготовить **6–10 маркеров** лута (наклейки / монетки)
- [ ] Зелёная лента или бумага для **экстракта**
- [ ] Таймер на телефоне: **12:00**

---

## За 15 минут до старта

### Разметка

- [ ] Отмерить и обозначить границы 4×4 м
- [ ] Наклеить **спавны** в 3–4 углах
- [ ] Наклеить **экстракт** 1×1 м по центру южной стороны (y: 3–4 м)

### Расстановка кубиков (см. `01-map-layout.md`)

- [ ] **Центр:** крест 5–7 кубиков
- [ ] **Восток:** длинная стена 4–5 кубов + окно
- [ ] **Запад:** стопка ящиков 3+1
- [ ] **Север:** L-укрытие
- [ ] **Проход:** 2 куба коридором юго-запад
- [ ] **Экстракт:** 2 низких куба по бокам «ворот»
- [ ] Проверить проходы **≥ 40 см**

### Лут

- [ ] Разложить маркеры на полу по схеме
- [ ] Перемешать стопки карточек (обычная / ценная)
- [ ] (Опц.) Под 1 куб без маркера — золотая карточка
- [ ] (Опц.) «Закрытая» точка: ключ + 1 золото в коробочке

### Игроки

- [ ] Каждому: 3 карточки «пистолетный магазин», 1 бинт
- [ ] Объяснить: 4 предмета макс, 2 ранения = смерть
- [ ] Выбрать режим: **пошаговый** или **realtime**
- [ ] Договориться про Scav: да / нет

---

## Старт раунда

1. [ ] Игроки на спавнах, спиной к центру или лицом по договорённости
2. [ ] «Рейд начался» — запуск таймера
3. [ ] За **2 минуты** до конца объявить: «Scavs приближаются» (опционально +1 Scav)
4. [ ] Таймер 0:00 — кто не в экстракте, **MIA** (лут на полу)

---

## После раунда

- [ ] Выжившие переносят лут в **тайники** (коробки с именами)
- [ ] Мёртвые забирают только «страховку» (опц.: 1 монета)
- [ ] Фото карты / быстрая перестановка 2–3 кубов — новый раунд
- [ ] Записать шутку раунда в угол README, если хотите

---

## Быстрая шпаргалка на стол

```
ДВИЖЕНИЕ: 1 м = 1 действие
СТРЕЛЬБА: d6 ≥ 3/4/5 (ближ/сред/даль)
УКРЫТИЕ: −1 к попаданию
3 выстрела = магазин
2 рана = смерть
ЭКСТРАКТ: зелёная зона, 10 сек / пропуск хода
ЛУТ: d6 на маркере, макс 4 вещи
```

## Что докупить / достроить потом

| Улучшение | Зачем |
|-----------|-------|
| Ещё 20 кубиков | Вторая этажность, мостики |
| Красная лента | «Кровь» / опасная зона |
| Фонарики | Режим ночь в тёмной комнате |
| Маски / повязки | Не смотреть в глаза — хардкор |
| Вторая комната 4×4 | Рейд с переходом через «дверь» |
